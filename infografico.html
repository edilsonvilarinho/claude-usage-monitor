Seção 6
6 Estados Especiais da Interface
⏸
Rate Limit
API retornou 429.
Faixa laranja com contagem regressiva.
triggerNow() bloqueado até expirar.
Backoff: 5→10→20→40→60 min.
❌
Erro de API
Falha na requisição (não 429).
Banner vermelho visível na UI.
Retry apenas em erros 5xx;
4xx (exceto 429) não retenta.
🔄
Reiniciando
resets_at está no passado.
Exibe "Reiniciando..." até o
próximo update da API confirmar
os novos valores da janela.
