📋 Referência de Códigos

Atributos Data

  • data-i18n="key" - Tradução de texto
  • data-i18n-title="key" - Tradução de title/tooltip
  • data-i18n-opt="key" - Tradução de option
  • data-tab="tab-id" - Aba do settings
  • data-cost-tab="session|weekly|monthly" - Aba de custo

Classes Utilitárias

  • .hidden - Oculto (display: none)
  • .active - Estado ativo
