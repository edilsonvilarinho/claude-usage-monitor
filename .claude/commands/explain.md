# /explain — Explicação em 1 Frase

Explica código em uma frase curta.

> Use para: dúvidas rápidas sem contexto longo

**Quando usar:**
- "Pra que serve essa função?" → `/explain parseCredentials`
- "O que essa lógica faz?" → `/explain computeSmartStatus`
- "Para que esse campo?" → `/explain rateLimitedUntil`

**Exemplo de input:**
```
/explain diffMinutesToReset
```

**Exemplo de output:**
```
Calcula minutos restantes até o próximo reset da sessão.
```
